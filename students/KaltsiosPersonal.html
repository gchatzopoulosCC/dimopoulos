How I created this personal webpage

In this personal webpage i have used the section tag to divide the layout into title, text, social-media and a table of websites for hobbies. This is done just so I can edivide the section and work on them seperately on the css file. I also use the div tag so I can seperate the parts between the section I want to style differently. The h1 and h2 tags were used for headings in the paragraphs. I also created an unordered list so I could list the pages I created using ul and li tags. Below you will find a table in which I have embeded some websites related to my hobbies At last, I have made a "find me on instagram" row in which I have linked my social media accounts to their respective icons.
